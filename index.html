第1回 WWW と HTML

今回はWebページの構造を記述するための書式HTMLの講義でした。

HTML内のスペースや改行は全て一つの半角スペースとして認識されるため改行して表示したい場合はpで囲わないといけないことを学びました。

また見出しなどで文字を目立たせたい場合はh1-6で囲むことも覚えた。数字が大きくなるほど文字の大きさも大きくなるように感じるが数字が小さい方が文字の大きさが大きいのでそれに気をつけたいです。

  • 演習1-4:HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

今回はウェブページの装飾をするためのCSSの書き方の講義

htmlとCSSを作りそれらを結びつけ装飾をしていく。

CSSでバックグラウンドと入力すると背景の色を変えることができ、テキストシャドウと入力すると文字に影をつけることができる。

選択した範囲を四角に囲み格子状に整えることも可能である。

  • ウェブブラウザの紹介

第3回 JavaScript の基本

今回はjavaScriptについて学んだ。

JavaScriptはWebページにおいて複雑な機能を表すプログラムである。

JavaScriptとjavaは名前が似ているが全く違うもの。

javaScriptを使用するとマウスのカーソルに合わせて色を変えることも可能になる。

  • オブジェクトの参照

第4回 DOM と JavaScript

DOMについて学んだ

DOMとはWebコンテンツを表現するデータ構造である。これを使用して内容を読み取ったりコンテンツに変更を加えることができる。

DOMを使用するとHTMLを変更することなく画像を追加したり、項目の削除を行うことができた。

  • 練習問題4-2~4-5

第5回 イベント処理

イベント処理を学んだ。

キーボードをタップしたら色を変えるようにしたりすることができるようになる。

またインプット要素についても学んだ。
